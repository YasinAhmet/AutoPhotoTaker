# AutoPhotoTaker
This program takes screenshots automatically and uses them like a video player.

You can enter the absolute path into the text panel that stays in the left side of the "Play video" button. The absolute path should be like that: G:\\Program
After that, you will be able to start the program. You can click to "Start/stop" button on the left and wait for a few seconds. Then you can click again to stop the program.
Now you are able to use "Play video" feature without errors. You can click the "Play video" button to start watching. The video will end after there is no more pictures.
